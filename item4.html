DASHBOARD

Se creó un dashboard interactivo en Power BI utilizando datos de ventas de Amazon para analizar el rendimiento del negocio. Incluye indicadores clave como ventas totales, unidades vendidas, devoluciones y ventas por segmento, permitiendo una visión clara de los patrones de compra y el desempeño por mercado. Esta herramienta facilita la toma de decisiones estratégicas basadas en datos.

OBJETIVO

Diseñar un dashboard interactivo en Power BI que permita visualizar y analizar el rendimiento de ventas de Amazon, proporcionando insights clave.

METODOLOGIA

Metodología:Se recopilaron y estructuraron datos de ventas, que fueron analizados para identificar patrones y tendencias. Posteriormente, se diseñaron visualizaciones con indicadores clave.

RESULTADO

El dashboard permite identificar patrones de compra, productos más vendidos y mercados rentables, facilitando la toma de decisiones basada en datos.
